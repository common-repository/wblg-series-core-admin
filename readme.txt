== Description ==

A plugin administration core essential for all plugins in the series.

== Installation ==


1. Use the WordPress Add New Plugins menu otherwise...
2. Download the zip file from WordPress.org.
3. Upload the *wblg-series-core-admin* folder to */wp-content/plugins/*. 1, Extract the zip file.
4. Activate the plugin through the WordPress plugins page.
5. See in Menu Admin Side Bar: WbLgnd Series.

== Frequently asked questions ==
 
This plugin do nothing useful in front-end. It manages the administration user interface by providing a unified presentation, some scripts shared and a single credit, donation and news page for plugins in the WebLegend series.

For an overview of the project WebLegend series, go to the [official website](http://www.la-dame-du-lac.com/ "WebLegend series on la-dame-du-lac.com") homepage of the ongoing development.
